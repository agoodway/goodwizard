## ADDED Requirements

### Requirement: Memory Seeds creates episodic and procedural directories

`Goodwizard.Memory.Seeds.seed/1` SHALL create the `memory/episodic/` and `memory/procedural/` subdirectories within the workspace when they do not exist.

#### Scenario: Seed creates episodic directory

- **WHEN** `Memory.Seeds.seed/1` is called with a workspace path
- **AND** the `memory/episodic/` directory does not exist
- **THEN** the directory SHALL be created
- **AND** the function SHALL return `:ok`

#### Scenario: Seed creates procedural directory

- **WHEN** `Memory.Seeds.seed/1` is called with a workspace path
- **AND** the `memory/procedural/` directory does not exist
- **THEN** the directory SHALL be created
- **AND** the function SHALL return `:ok`

#### Scenario: Seed is idempotent for directories

- **WHEN** `Memory.Seeds.seed/1` is called with a workspace path
- **AND** the `memory/episodic/` and `memory/procedural/` directories already exist
- **THEN** the function SHALL NOT raise an error
- **AND** the function SHALL return `:ok`

---

### Requirement: Memory Seeds ensures MEMORY.md exists

`Goodwizard.Memory.Seeds.seed/1` SHALL create `memory/MEMORY.md` as an empty file if it does not already exist, and SHALL NOT overwrite existing content.

#### Scenario: Seed creates MEMORY.md when missing

- **WHEN** `Memory.Seeds.seed/1` is called with a workspace path
- **AND** `memory/MEMORY.md` does not exist
- **THEN** the file SHALL be created with empty content

#### Scenario: Seed preserves existing MEMORY.md

- **WHEN** `Memory.Seeds.seed/1` is called with a workspace path
- **AND** `memory/MEMORY.md` already exists with content "User prefers dark mode"
- **THEN** the file content SHALL remain "User prefers dark mode"

---

### Requirement: Setup task calls Memory Seeds

`mix goodwizard.setup` SHALL call `Goodwizard.Memory.Seeds.seed/1` during workspace initialization so that new workspaces include the episodic and procedural memory directories.

#### Scenario: Fresh workspace setup creates memory subdirectories

- **WHEN** `mix goodwizard.setup` is run on a new workspace
- **THEN** the `memory/episodic/` directory SHALL exist
- **AND** the `memory/procedural/` directory SHALL exist
- **AND** the `memory/MEMORY.md` file SHALL exist

---

### Requirement: Preamble describes three-memory architecture

The system prompt preamble generated by `Goodwizard.Character.Preamble.generate/0` SHALL describe the three memory types (semantic, episodic, procedural) with their storage locations and usage guidance.

#### Scenario: Preamble contains semantic memory description

- **WHEN** `Preamble.generate/0` is called
- **THEN** the result SHALL contain text describing semantic memory
- **AND** the result SHALL reference `brain/` and `memory/MEMORY.md` as storage locations

#### Scenario: Preamble contains episodic memory description

- **WHEN** `Preamble.generate/0` is called
- **THEN** the result SHALL contain text describing episodic memory
- **AND** the result SHALL reference `memory/episodic/` as the storage location
- **AND** the result SHALL describe episodes as records of past experiences

#### Scenario: Preamble contains procedural memory description

- **WHEN** `Preamble.generate/0` is called
- **THEN** the result SHALL contain text describing procedural memory
- **AND** the result SHALL reference `memory/procedural/` as the storage location
- **AND** the result SHALL describe procedures as learned behavioral patterns with confidence levels

#### Scenario: Preamble stays within size limit

- **WHEN** `Preamble.generate/0` is called
- **THEN** the returned string SHALL be less than 10,240 bytes
